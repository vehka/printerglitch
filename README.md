# printerglitch

A p5.js script that creates images that are inspired by glitchy printers and especially [this post](https://llllllll.co/t/glitch-art/12598/100) by Whinger.

Open the printerglitch.html in your browser and experiment. Large canvas sizes might be slow to use.

Created over a weekend in December 2025 by vehka. Lots of help from Gemini & Antigravity + Claude Sonnet 4.5 & Claude Code.
